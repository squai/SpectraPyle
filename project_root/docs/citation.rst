Citation & Acknowledgments
==========================

If you use SpectraPyle in your research, please cite the following paper
and include the acknowledgment text below.

How to cite
-----------

Primary reference:

   Euclid Collaboration, Quai S., Pozzetti L., et al. 2025,
   *Astronomy & Astrophysics*, DOI: `10.1051/0004-6361/202557329
   <https://doi.org/10.1051/0004-6361/202557329>`_

**BibTeX:**

.. code-block:: bibtex

   @misc{euclidcollaboration2025spectraPyle,
         title={Euclid preparation. Predicting star-forming galaxy scaling
                relations with the spectral stacking code SpectraPyle},
         author={Euclid Collaboration and S. Quai and L. Pozzetti and M. Talia
                 and C. Mancini and P. Cassata and others},
         year={2025},
         eprint={2509.16120},
         archivePrefix={arXiv},
         primaryClass={astro-ph.GA},
         doi={10.1051/0004-6361/202557329},
         url={https://arxiv.org/abs/2509.16120}
   }

**APA:**

   Euclid Collaboration, Quai, S., Pozzetti, L., Talia, M., Mancini, C.,
   Cassata, P., et al. (2025). *Euclid preparation. Predicting star-forming
   galaxy scaling relations with the spectral stacking code SpectraPyle*.
   Astronomy & Astrophysics. https://doi.org/10.1051/0004-6361/202557329

Instrument-specific acknowledgments
------------------------------------

If you use SpectraPyle with **Euclid** data, you must also follow the
`Euclid Consortium publication policy
<https://www.euclid-ec.org/>`_ and include the standard Euclid acknowledgment
text as specified in the consortium guidelines.
